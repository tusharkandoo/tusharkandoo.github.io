# Prime Video Clone

This project is a front-end clone of the Amazon Prime Video website developed as part of a college web development assignment.  
The objective of this project is to understand and implement responsive web design using only HTML and CSS.

---

# Objective

- To design a responsive website layout
- To apply HTML5 semantic elements
- To practice CSS styling and layout techniques
- To understand real-world UI structure

---

# Technologies Used

- HTML5
- CSS3
- GitHub Pages (for deployment)



# Features

- Responsive layout for different screen sizes
- Navigation bar similar to Prime Video
- Movie and series card layout
- Hover effects using CSS
- Clean and structured code
